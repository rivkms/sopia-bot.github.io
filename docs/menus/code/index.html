소피아의 모든 동작을 프로그래밍

1. 진입점

이 문서에선 CODE 탭의 설명만 진행합니다.
상세한 프로그래밍 방법은 API DOC 을 참고하세요.
소피아가 초기화 될 땐 무조건 sopia/main.js 파일을 실행시킵니다. 즉, main.js는 무조건 필요합니다.

2. 에디터 사용 방법

소피아의 에디터는 강력한 VS Code 의 기반인 Monaco Editor 를 사용하고 있습니다. 그 기능은 웬만한 에디터보다 강력합니다.

저장하기

Ctrl + S 키를 눌러서 저장할 수 있습니다. 정상적으로 저장되면 위와 같은 알림창이 왼쪽 아래에 올라옵니다.

저장에 실패하면 위와 같은 알림창이 뜹니다. 이 경우, 문법이 잘못된 것이니 수정을 잘 했나 확인해 보십시오.

리로딩

F5 키를 눌러 코드를 적용시킬 수 있습니다. 매니저 실행 도중 수정한 내용을 즉시 적용해 보세요.
